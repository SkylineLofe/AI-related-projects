This could have been improved by deepening my multivariate analysis with additional plots (for example, I could have added in point-points), and I could have cleaned up any lingering error outputs that existed while I was working and completing the project.

In order to do this, I had to analyze the provided data to understand key customer attributes influencing loan acquisition. In doing so, I built a predictive model that captured patterns and customer characteristics, which would help the bank effectively target potential loan buyers, improving marketing efforts and increasing conversion rates. For this project, I felt like I did a fairly good job at my first attempt.
